{:navgoco}

Navgoco is a simple JQuery plugin which turns a nested unordered list of links into a beautiful vertical multi-level sliding navigation with options to remember sub-menus visible state between sessions or act as an accordion.

Download Github

Demo: Remember state

    Collapse All | Expand All

    Javascript

    $(document).ready(function() {
	// Setup navgoco menu with default options
	$("#demo1").navgoco({
		caret: '<span class="caret"></span>',
		accordion: false,
		openClass: 'open',
		save: true,
		cookie: {
			name: 'navgoco',
			expires: false,
			path: '/'
		},
		slide: {
			duration: 400,
			easing: 'swing'
		}
	});
});

    Demo: Accordion

      Javascript

      $(document).ready(function() {
	// Setup navgoco accordion menu, cookie save is turned off automatically
	$("#demo1").navgoco({accordion: true});
});
